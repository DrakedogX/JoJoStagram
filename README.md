# 프로젝트명
> 코틀린으로 인스타그램 클론앱 만들기

개인 토이 프로젝트로써 Kotlin과 Firebase으로 동작하는 Instagram Clone App 입니다.

개발은 http://www.yes24.com/24/goods/62495788 해당를 도서 보고 참조했습니다.

원본 소스 주소는 다음과 같습니다.

https://github.com/you6878/howlstagram

좋은 내용으로 책을 출간해주신 하울님께 감사드립니다.
